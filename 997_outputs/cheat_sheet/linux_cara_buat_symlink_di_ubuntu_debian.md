**Format**

```bash
ln -s [path_target_asli] [nama_symlink]
```

**Cek Symlink**

```bash
ls -l
```

**Hapus Symlink**

```bash
rm [file]
rm -r [folder]
```

Hanya menghapus Symlink saja, tidak menghapus sumbernya.
