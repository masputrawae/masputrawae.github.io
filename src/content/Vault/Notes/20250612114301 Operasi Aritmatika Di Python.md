# Operasi Aritmatika Di Python

Operasi Aritmatika adalah Operasi Matematika Dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian, untuk menghitung sebuah nilai.

Aritmatika dasar di Python:

- Penjumlahan `+`
- Pengurangan `-`
- Perkalian `*`
- Pembagian `/`
- Sisa Bagi (modulo) `%`
- Eksponen (pangkat) `**`
- Pembagian Bulat `//`

**Contoh**:

```python
a = 10
b = 7

print(a + b)
print(a - b)
print(a * b)
print(a / b)
print(a % b)
print(a // b)
print(a ** b)
```
