# Dictionary Di Python
Struktur data dalam bentuk pasang key value
```python
siswa = {
	'nama': 'John Doe',
	'umur': 27,
	'jurusan': 'Informatika'
}

print(siswa[nama]) # Output: John Doe
```

- Bisa mengubah, menambah, dan menghapus item
- key harus unik
