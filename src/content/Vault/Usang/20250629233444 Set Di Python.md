# Set Di Python

Kumpulan item yang unik dan tidak berurutan 

```python
angka = {1, 2, 3, 2}
print(angka)  # Output: {1, 2, 3}
```

- Tidak bisa diakses dengan index
- Cocok untuk operasi himpunan: Unio, Intersection, dll
