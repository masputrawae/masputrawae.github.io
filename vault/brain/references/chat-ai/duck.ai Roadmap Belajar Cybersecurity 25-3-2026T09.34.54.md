# Roadmap Belajar Cybersecurity

**Cybersecurity adalah bidang yang luas dan terus berkembang.** Berikut adalah roadmap terstruktur untuk memulai dan mengembangkan karir di bidang ini, dari tingkat pemula hingga advanced.

---

## Fase 1: Fondasi (0-3 bulan)

### Dasar-dasar Networking dan Sistem Operasi

Anda perlu memahami bagaimana jaringan dan sistem bekerja sebelum mempelajari cara mengamankannya.

- **Networking**: Pelajari model OSI, TCP/IP, DNS, HTTP/HTTPS, firewall, dan routing dasar
- **Sistem Operasi**: Kuasai Windows, Linux, dan macOS. Linux sangat penting karena banyak digunakan dalam security
- **Command Line**: Mahir menggunakan terminal/command prompt dan bash scripting

**Resources**: Coursera, Udemy, YouTube channels seperti Professor Messer

---

## Fase 2: Fondasi Cybersecurity (3-6 bulan)

### Konsep Dasar Security

Pelajari prinsip-prinsip inti cybersecurity dan terminologi industri.

- **CIA Triad**: Confidentiality, Integrity, Availability
- **Threat Modeling**: Memahami jenis-jenis ancaman (malware, phishing, ransomware, dll)
- **Cryptography Basics**: Enkripsi, hashing, digital signatures
- **Authentication & Authorization**: Passwords, multi-factor authentication, access control
- **OWASP Top 10**: Kerentanan aplikasi web yang paling umum

**Sertifikasi Target**: **CompTIA Security+** (entry-level yang sangat diakui)

---

## Fase 3: Spesialisasi Awal (6-12 bulan)

Pilih jalur yang sesuai dengan minat Anda. Ada beberapa arah yang bisa diambil:

### Jalur 1: Network & Infrastructure Security

- Firewall configuration
- Intrusion Detection/Prevention Systems (IDS/IPS)
- Virtual Private Networks (VPN)
- Network segmentation

**Sertifikasi**: **CompTIA Network+**, **Cisco CCNA Security**

### Jalur 2: Web Application Security

- OWASP Top 10 secara mendalam
- SQL Injection, Cross-Site Scripting (XSS), CSRF
- Secure coding practices
- API security
- Web application firewalls (WAF)

**Tools**: Burp Suite, OWASP ZAP

### Jalur 3: Ethical Hacking & Penetration Testing

- Reconnaissance dan scanning
- Exploitation techniques
- Post-exploitation
- Privilege escalation
- Social engineering

**Sertifikasi**: **CEH (Certified Ethical Hacker)**, **eJPT**

### Jalur 4: Cloud Security

- AWS, Azure, GCP security
- Container security (Docker, Kubernetes)
- Infrastructure as Code (IaC) security
- Cloud compliance dan governance

---

## Fase 4: Pengalaman Praktis (Berkelanjutan)

### Hands-On Practice

- **HackTheBox**: Platform untuk praktik penetration testing
- **TryHackMe**: Pembelajaran interaktif dengan challenges
- **PortSwigger Web Security Academy**: Gratis dan komprehensif untuk web security
- **DVWA (Damn Vulnerable Web Application)**: Aplikasi yang sengaja vulnerable untuk latihan
- **Capture The Flag (CTF)**: Kompetisi security

### Homelab

Bangun lab sendiri di rumah untuk bereksperimen:

- Virtual machines (VirtualBox, VMware)
- Setup network lokal dengan firewall
- Deploy vulnerable applications untuk practice

---

## Fase 5: Pengembangan Lanjutan (12+ bulan)

### Spesialisasi Mendalam

Setelah memiliki fondasi kuat, pilih spesialisasi:

- **Incident Response & Forensics**: Menangani breach dan investigasi
- **Threat Intelligence**: Analisis ancaman dan intelligence gathering
- **Security Architecture**: Merancang sistem security yang robust
- **DevSecOps**: Mengintegrasikan security dalam development pipeline
- **Malware Analysis**: Analisis malicious code
- **Governance, Risk & Compliance (GRC)**: Compliance frameworks seperti ISO 27001, NIST

**Sertifikasi Lanjutan**: **CISSP**, **OSCP**, **GIAC certifications**, **CEH Master**

---

## Fase 6: Karir & Pertumbuhan Berkelanjutan

### Jalur Karir Potensial

- **Security Analyst**: Monitoring dan analysis ancaman
- **Penetration Tester**: Testing keamanan sistem klien
- **Security Engineer**: Merancang dan mengimplementasikan solusi security
- **Security Architect**: Merancang strategi security enterprise
- **Incident Response Specialist**: Menangani security incidents
- **Security Consultant**: Memberikan advice kepada organisasi

### Pembelajaran Berkelanjutan

- Ikuti industry certifications terbaru
- Ikuti webinar dan konferensi (DEF CON, Black Hat, RSA Conference)
- Baca security blogs dan research papers
- Bergabung dengan komunitas (OWASP, security meetups)
- Kontribusi ke open-source security projects

## Tips Sukses

1. **Mulai dari dasar**: Jangan skip fondasi networking dan sistem operasi
2. **Praktik konsisten**: Teori saja tidak cukup—hands-on practice adalah kunci
3. **Bangun portfolio**: Dokumentasikan project dan learning Anda
4. **Networking**: Bangun hubungan dengan profesional di bidang ini
5. **Tetap update**: Cybersecurity terus berkembang, ikuti perkembangan terbaru
6. **Sabar**: Ini adalah journey jangka panjang, tidak ada shortcut

Roadmap ini fleksibel—sesuaikan dengan kecepatan dan minat Anda. Fokus pada satu area terlebih dahulu sebelum meluas.
