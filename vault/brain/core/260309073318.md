Tanpa pertanyaan seseorang tidak akan pernah belajar. Karena pemicu seseorang untuk berfikir adalah pertanyaan, fase yang logis seperti ini

```
tidak tahu → sadar tidak tahu → bertanya → mencari → memahami
```

Jika kita berhenti di pada `sadar tidak tahu` tanpa ada apa, kenapa, dan bagaimana, kemungkinan kita akan berfikir ya udah next aja.

> Makanya aku setuju dengan peribahasa malu bertanya sesat di jalan
