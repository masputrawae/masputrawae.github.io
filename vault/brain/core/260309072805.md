Kesuksesan tidak selalu tentang uang atau kekayaan. Namun sukses itu bisa berupa banyak hal.

- Berhasil mendapatkan nilai bagus saat ujian => Sukses
- Bisa memecahkan masalah dalam pekerjaan => Sukses
- Memahami Konsep dasar pemrograman => Sukses
- Menyelesaikan puzzle => Sukses

Super fleksibel, intinya apapun keberhasilan dalam hidup, entah itu kecil ataupun besar, itu bisa kita anggap sebagai kesuksesan
