Program pertama di bahasa c

**Buat File**

```bash
vim hello.c
```

**Isi Dengan**

```c
#include <stdio.h>

int main(){
    printf("Hello, World!\n");
    return 0;
}
```

**Simpan**

```vim
:wq
```

**Compile program**

```bash
ggc hello.c -o hello
```

**Jalankan**

```bash
./hello
```
