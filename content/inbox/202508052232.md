Di dalam bahasa C, titik koma itu wajib. Contoh 
```c
int x = 10;
printf("Nilai x: %d\n", x);
```

Kalau lupa 
```c
int x = 10
printf("Nilai x: %d\n", x);
```

Maka saat di-*compile*, akan muncul error seperti ini
```bash
error: expected ‘;’ before ‘printf’
```
