# Lumos Theme Hugo

![Lumos Theme](https://picsum.photos/1920/720)

Tema Hugo modern untuk Knowledge Base, Digital Garden, dan dokumentasi dengan berbagai fitur canggih.

## Fitur Utama

### Tampilan & Responsivitas
- **Desain Responsif** dengan 3 breakpoint:
  - 3 kolom (desktop)
  - 2 kolom + 1 Offcanvas (tablet)
  - 1 kolom + 2 Offcanvas (mobile)
- **Mode Tema**: Dark - Auto - Light
- **Font Family**: Noto Sans, Noto Serif, Noto Sans Mono

### Navigasi & Konten
- **Menu Hierarkis** (Tree View Navigation)
- Menu default (opsional)
- **Daftar Isi** (Table of Contents) dengan highlight
- **Wikilink Support**:
  - Deteksi Broken Link
  - Deteksi External Link
  - Embed Image
  - Link Preview Popup Hover (seperti Obsidian)

### Fitur Teknis
- **Multilingual Support** (dukungan multi bahasa)
- **Pencarian** dengan integrasi Fuse.js
- **Komentar** dengan Giscus
- **Diagram**:
  - Mermaid Diagrams
  - Goat Diagram
- **Historis** menggunakan gitInfo
- **PWA Support** (Progressive Web App) bawaan
- **SEO Friendly** (optimasi untuk mesin pencari)

### Kategori & Konten
- **Taxonomy** default: tags, categories
- **Shortcodes** bawaan untuk berbagai kebutuhan

## Tujuan Penggunaan

Tema ini dirancang khusus untuk:
- Knowledge Base
- Digital Garden
- Dokumentasi
- Catatan Pemikiran
- Pembelajar

## Instalasi & Penggunaan

1. Tambahkan tema ke project Hugo Anda:
   ```bash
   git submodule add https://github.com/masputraw/lumos_theme themes/lumos
   ```

2. Atur tema di `config.toml`:
   ```toml
   theme = "lumos"
   ```

3. Lihat panduan lengkap di [Dokumentasi](/guide/)
