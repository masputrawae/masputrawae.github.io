**Tema canggih untuk membangun Knowledge Base dan Digital Garden**

## Fitur Unggulan

### Knowledge-Oriented
Dirancang khusus untuk pengelolaan pengetahuan dengan dukungan Wikilink, dan pencarian canggih

### Dark/Light Mode
Dukung tema gelap/terang dengan toggle otomatis berdasarkan preferensi sistem

### Pencarian Instant
Integrasi Fuse.js untuk pencarian cepat di seluruh konten

### Multi Bahasa
Dukungan penuh untuk situs multilingual

### Diagram & Visualisasi
Dukung Mermaid.js dan Goat Diagram untuk visualisasi pengetahuan

### PWA Support
Bisa diinstall sebagai aplikasi mobile (Progressive Web App)

## Mulai Cepat

1. Tambahkan tema ke project Hugo Anda:
   ```bash
   git submodule add https://github.com/masputraw/lumos_theme themes/lumos
   ```

2. Set tema di `config.toml`:
   ```toml
   theme = "lumos"
   ```

3. Jalankan server:
   ```bash
   hugo server -D
   ```
