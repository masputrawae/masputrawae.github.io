Implikasi logika Matematika

Bentuk kalimat:
A → B artinya:

> Jika A benar, maka B juga harus benar

Contoh:

- A: "Saya Belajar"
- B: "Saya Lulus Ujian"

A → B = "JIKA saya belajar, MAKA saya lulus ujian"

Aturannya:

| A (Premis) | B (Konklusi) | A → B |
| ---------- | ------------ | ----- |
| True       | True         | True  |
| True       | False        | False |
| False      | True         | True  |
| False      | False        | True  |

> [!Question]
> Kenapa 3 baris terakhir itu True?
> Karena implikasi hanya salah kalau A benar tapi B-nya gagal.

Misalnya:
“Kalau saya belajar (A = true), tapi saya tetap gak lulus (B = false)” → ini berarti janji/implikasinya gagal, maka hasilnya false.
