Bahasa pemrograman yang tidak perlu di kompilasi ke dalam bentuk biner atau bahasa mesin. Baris kode akan dieksekusi langsung dari file sumber baris demi baris oleh program yang bernama _Interpreter_.

**Contoh**:
- JavaScript
- Python
- Ruby
- PHP
- Perl

Umumnya Bahasa Pemrograman Interpretatif cenderung sedikit lebih lambat di bandingkan bahasa pemrograman kompilatif.
